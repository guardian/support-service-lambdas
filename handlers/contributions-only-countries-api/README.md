# Contributions only countries API

This API is an API Gateway triggered lambda to return the list of
countries in which we will only sell contributions, not subscription products. 
It is used by [support-dotcom-components](https://github.com/guardian/support-dotcom-components) and [support-frontend](https://github.com/guardian/support-frontend) to ensure that users from the countries listed are not able to purchase any products besides contributions.

It is a public API endpoint

# Urls

- PROD: https://contributions-only-countries-api.support.guardianapis.com/contributions-only-countries
- CODE: https://contributions-only-countries-api-code.support.guardianapis.com/contributions-only-countries
